_EMAIL CONFIG_
```toml
[email]
email_backend = airflow.utils.email.send_email_smtp
smtp_host = smtp.gmail.com
smtp_starttls = True
smtp_ssl = False
smtp_user = your-email@gmail.com
smtp_password = your-email-password
smtp_port = 587
smtp_mail_from = your-email@gmail.com

```
